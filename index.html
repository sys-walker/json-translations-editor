About

This is the first language editor (at least as far as I know) from @ngx-translate/core.

The motivation that made me create this editor was my constant problem with the internationalization of applications.

The @ngx-translate/core plugin uses JSON for translations, but creating JSONs can be a difficult task if you are dealing with many languages or if you have to modify translations.

At that moment I remembered that Android Studio has a language editor to create XML translation files, facilitating the process of internationalizing Android applications.

That's why I designed this editor.

Why is there only one file and is it so bloated?

To avoid dependencies with external libraries and to be completely offline. Also because the moment you move the editor to another folder you don't have to worry about whether it about the dependencies
